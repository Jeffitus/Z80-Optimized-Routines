String Search, Replace, and Delete
By Weregoose
Inputs:
Str0= Original string
Str1= String to search for
Str2= Replace all occurances of Str1 in Str0 with this, for deletion make the length of Str2 0
Output: Ans and Str3 is the resulting string

:length(Str1→U
:length(Str2→V
:"?"+Str0+"?→Str0
:inString(Ans,Str1,2→W
:While Ans
:sub(Str0,1,W-1
:If V:Ans+Str2
:Ans+sub(Str0,U+W,length(Str0)-U-W+1→Str0
:inString(Ans,Str1,V+W→W
:End
:sub(Str0,2,length(Str0)-2→Str3
